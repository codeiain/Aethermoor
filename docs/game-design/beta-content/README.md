Beta introduces:

- 3 new zones expanding the world beyond the Alpha seed area
- Live MMO social systems: parties, trade, chat
- Combat balance per class at tiers 1–5, 6–10, 11–15, 16–20

All content designed to hand off directly to CTO for implementation.
