# Aethermoor — Starter Zone Art Assets

All files are **RGBA PNG** with transparency. Generated by
`frontend/scripts/generate_assets.py` (stdlib only, no third-party deps).

## Directory layout

```
frontend/src/assets/
  tiles/     — 16x16 px terrain & object tiles
  sprites/   — character spritesheets (see dimensions below)
  ui/        — HUD, inventory, and button elements
```

## Character Spritesheets

### Player heroes — 64×64 px

Layout: **4 columns (animation frames 0–3) × 4 rows (directions)**

| Row | Direction |
|-----|-----------|
| 0   | Down (toward camera) |
| 1   | Up (away from camera) |
| 2   | Left |
| 3   | Right |

Each frame cell is **16×16 px**.

| File | Character |
|------|-----------|
| `hero_m.png` | Player — Fighter Male (red shirt, blue trousers) |
| `hero_f.png` | Player — Fighter Female (blue shirt, brown trousers) |

**Phaser.js load example:**
```js
this.load.spritesheet('hero_m', 'assets/sprites/hero_m.png', { frameWidth: 16, frameHeight: 16 });
// Frame index = direction_row * 4 + animation_frame
```

### NPCs — 64×16 px

Layout: **4 columns (directions: down/up/left/right) × 1 row (idle)**

Each frame cell is **16×16 px**.

| File | NPC |
|------|-----|
| `npc_farmer.png` | Farmer (green shirt) |
| `npc_blacksmith.png` | Blacksmith (dark apron) |
| `npc_innkeeper.png` | Innkeeper (tan tunic) |

**Phaser.js load example:**
```js
this.load.spritesheet('npc_farmer', 'assets/sprites/npc_farmer.png', { frameWidth: 16, frameHeight: 16 });
// Frame 0=down, 1=up, 2=left, 3=right
```

### Enemy — 64×32 px

Layout: **4 columns × 2 rows**

| Row | Animation |
|-----|-----------|
| 0   | Walk cycle — 4 frames (down-facing) |
| 1   | Attack cycle — 4 frames (down-facing, weapon drawn) |

Each frame cell is **16×16 px**.

| File | Enemy |
|------|-------|
| `enemy_goblin.png` | Goblin Scout (green skin, red eyes) |

## Regenerating assets

```bash
python3 frontend/scripts/generate_assets.py
```

No dependencies beyond Python 3 stdlib. All PNGs are RGBA colour type 6.
Suitable for Phaser.js `load.image()` and `load.spritesheet()`.
